## 🚀 快速開始

### 系統需求

| 項目 | 版本 |
|------|------|
| Python | 3.10 以上 |
| 作業系統 | Windows / macOS / Linux |
| 瀏覽器 | Chrome / Edge / Firefox（任一） |

### 安裝步驟

```bash
# 1. Clone 專案
git clone https://github.com/yhtung168/stock-analysis.git
cd stock-analysis

# 2. 建立虛擬環境（推薦）
python -m venv .venv

# 3. 啟動虛擬環境
# Windows (Git Bash):
source .venv/Scripts/activate
# Windows (PowerShell):
.venv\Scripts\Activate.ps1
# macOS / Linux:
source .venv/bin/activate

# 4. 安裝依賴套件
pip install -r requirements.txt

# 5. 啟動應用程式
streamlit run main.py
```

### 啟動成功畫面

```
  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:8501        ← 點這個開啟
  Network URL: http://192.168.x.x:8501   ← 同網路的其他裝置也能開
```

瀏覽器會自動打開，看到深色主題的分析平台就代表成功了！

### 📈 Tab 2: Technical — 技術分析

**功能：** 互動式 K 線圖 + 15 種技術指標

```
可疊加在 K 線圖上的指標：
  • 移動平均線（5/10/20/60/120/240日）
  • 布林通道 (Bollinger Bands)
  • 拋物線 SAR (Parabolic SAR)
  • 一目均衡表 (Ichimoku Cloud)

獨立子圖表指標：
  • MACD          • RSI (14)

  • OBV 量能指標   • Williams %R
  • CCI 順勢指標   • ATR 波動率
```

每個指標都附有**教學說明卡片**，解釋判讀方式。

### 📉 Tab 5: Market Compare — 市場比較

```
  • 📈 績效比較圖 — 以 100% 為基準，比較相對表現
```

### 🔄 Tab 6: Backtest — 回測

**功能：** 用歷史數據驗證交易策略是否有效

提供三種回測模式：

| 模式 | 說明 | 適合誰 |
|------|------|--------|
| **Strategy 策略** | 選擇預設策略（黃金交叉、RSI、MACD 等） | 新手入門 |
| **Score 評分** | 用綜合評分做進出場依據（買入/賣出門檻） | 中階使用者 |
| **Advanced 進階** | 自訂買賣條件（最多 10 個條件，AND/OR 邏輯） | 進階使用者 |

```
風險管理設定（三種模式共用）：
  • 停損 % (Stop-Loss)
  • 停利 % (Take-Profit)
  • 移動停損 % (Trailing Stop)
  • 資金比例 % (Position Size)
  • 初始資金（預設 100 萬）
```

**回測結果包含：**

```
  📈 圖表
  • 價格走勢圖 + 買賣訊號標記（🟢買 / 🔴賣）
  • 權益曲線 vs 買入持有比較
  • 最大回撤圖

  📊 績效指標（12 項）
  ┌──────────────┬──────────────┬──────────────┐
  │ 總報酬率      │ CAGR 年化    │ 最大回撤      │
  │ Sharpe Ratio │ Sortino      │ Calmar       │
  │ 勝率         │ 獲利因子      │ 期望值        │
  │ 平均獲利      │ 平均虧損      │ 總交易次數    │
  └──────────────┴──────────────┴──────────────┘

  📋 交易紀錄表
  • 每筆交易的進出場日期、價格、報酬率
```

### 📋 Tab 7: Batch Analysis — 批次分析

**功能：** 一次分析多檔股票，快速找出值得關注的標的

```
操作流程：
  1. 輸入多個股票代號（逗號或換行分隔）
     或一鍵載入「台股 Top 50」/「美股 Top 50」
  2. 點擊執行 → 等待分析完成（有進度條）
  3. 查看結果表格 → 匯出 Excel

結果表格欄位：
  股票代號 | 名稱 | 綜合評分 | 信號 | 亮點 | 風險

額外功能：
  • 🏆 Top Picks — 自動篩出評分 ≥ 7 的強勢股
  • 📥 Excel 匯出 — 包含所有指標詳細數據
```

---

### 🚀 Tab 8: Momentum — 動能篩選

**功能：** 找出近期最強勢和最弱勢的股票

```
操作流程：
  1. 選擇股票池（台股 Top 50 / 美股 Top 50 / 自訂）
  2. 選擇計算期間（1天/5天/1月/3月/6月/1年）
  3. 執行篩選

結果包含：
  • 📊 動能排行橫條圖（Top 30，紅綠色標示漲跌）
  • 📋 完整排名表格
    代號 | 名稱 | 報酬率 | 成交量變化 | RSI | 距高點% | 百分位 | 強弱
  • 🟢 強勢股清單 + 🔴 弱勢股清單
  • 📥 Excel 匯出
```

## 🔧 技術棧

| 類別 | 技術 | 用途 |
